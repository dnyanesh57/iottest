API Documentation

Upload Data (Meter → Server)

Endpoint: POST /upload

Content-Type: application/json

{
    "meter_id": "123456789012",
    "sensor_id": "28FF1234567890AB",
    "data": "000001 23-06-15 14:30:00 AA 25.50 2048"
}
Get Configuration (Meter ← Server)

Endpoint: GET /get_config/<meter_id>

Response:

{
    "ssid": "YourWiFiNetwork",
    "password": "WiFiPassword",
    "server_url": "http://your-server.com:5000",
    "sample_interval": 30
}
